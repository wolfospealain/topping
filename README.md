# TopPing
A top/uptime inspired version of ping.

```
Usage: topping.py [options] target

A top/uptime inspired version of ping: Average ping speeds for 1, 5, 15 min.
(statistics); distance (km); connection uptime / error time. Current ping
speed.

Options:
  --version             show program's version number and exit
  -h, --help            show this help message and exit
  -d, --distance        estimate distance in km with 2/3 lightspeed
  -p SECONDS, --pause=SECONDS
                        pause seconds between ping requests (default: 1)
  -s, --statistics      display minimum & maximum statistics
```
